
# Monitoring file system

This tool is used to record all activities in a specified folder, the tool will respond if data changes occur.

Contoh:
- New file 📄
- Edit file 📄
- Delete file 📄
- Move file 📄
- New folder 📁 
- Edit folder 📁
- Delete folder 📁
- Move folder 📁

# How to use
Start the server

```bash
  python app.py C:/
```
